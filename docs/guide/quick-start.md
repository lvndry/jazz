# Quick Start Guide

Get up and running with Jazz in minutes.

## 1. Install the CLI

Jazz is available via npm, bun, pnpm, or yarn.

```bash
# npm
npm install -g jazz-ai

# bun
bun add -g jazz-ai

# pnpm
pnpm add -g jazz-ai

# yarn
yarn global add jazz-ai
```

## 2. Chat With Your Agent

Simply run `jazz` to start a chat with your agent. Upon first run, Jazz will help you configure your initial settings.

```bash
jazz
```

## 3. Update Jazz

Keep Jazz up to date with the latest features and improvements:

```bash
jazz update
```
